# My Portfolio

This is a personal portfolio website built with React. It showcases my work, skills, and projects. This website is designed to demonstrate my abilities in front-end development and is hosted on GitHub Pages.

## Features

- **Home Page**: Overview of the portfolio.
- **About Me**: A section that includes details about my background, education, and skills.
- **Projects**: A showcase of projects I've worked on, with links to GitHub repositories and live demos.
- **Smooth Scrolling**: Smooth scrolling navigation between sections with a sleek animation.
- **Responsive Design**: The site is fully responsive, making it look great on both desktop and mobile devices.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **CSS & SCSS**: For styling and custom animations.
- **GitHub Pages**: For hosting the portfolio online.
